# WESEKAI ⚔️

WESEKAI is a dynamic, highly-curated anime recommendation engine powered by the **Wesley Intelligence Layer**. It specializes in surfacing the best and newest **Isekai, Fantasy, Military, Strategy, and Reincarnation** anime, complete with custom world-building scores and dynamic tagging.

## ✨ Features

- **The Intelligence Layer**: Automatically scans MyAnimeList (via the Jikan API) and dynamically tags anime based on synopsis keyword extraction and genre mapping.
- **World-Building Scoring**: Calculates a custom "World-Building Score" (out of 10) based on the depth of themes like politics, economy, kingdom-building, and magic systems.
- **Top 20 Grid View**: Fetches the most recent and relevant anime based on your selected filter, displaying them in a sleek, responsive grid.
- **The Arsenal (Watchlist)**: Save anime to your personal Arsenal. Saved anime are automatically hidden from the main discovery grid. Persists across sessions using `localStorage`.
- **Dropped List**: Banish anime you aren't interested in to the Dropped list, keeping your main feed clean. Persists via `localStorage`.
- **Thematic Filters**: Instantly pivot your recommendations between pure Fantasy, Isekai, Military, Strategy, or Reincarnation. (Strictly no Sci-Fi or Mecha!).
- **Resilient API Fetching**: Implements Exponential Backoff to gracefully handle Jikan API rate limits (429 errors) without breaking the user experience.
- **Highly Optimized**: Built with React performance best practices (`React.memo`, `useMemo`, `useCallback`, and native image lazy loading) to ensure buttery smooth animations even with large grids.

## 🛠️ Tech Stack & Architecture

- **Frontend Framework**: React 18 with TypeScript, powered by Vite.
- **Styling**: Tailwind CSS for a dark, glassmorphic, and highly polished UI.
- **Animations**: Framer Motion (`motion/react`) for fluid layout transitions, modal popups, and hover effects.
- **Icons**: Lucide React.
- **Data Source**: Jikan API v4 (Unofficial MyAnimeList API).

### Core Files
- `src/App.tsx`: The main application component containing the UI, state management (Arsenal/Dropped), and layout animations.
- `src/lib/mal.ts`: The data fetching layer. Handles Jikan API requests, exponential backoff, and the dynamic keyword extraction engine.
- `src/lib/scoring.ts`: Contains the logic for calculating the custom World-Building Score based on the tags generated by the Intelligence Layer.

## 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```
2. **Run the Development Server**
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## 💡 Usage

- **Scan for More**: Click the "Scan for More" button or any of the filter pills to fetch a fresh batch of 20 anime.
- **Save to Arsenal**: Click the Bookmark icon on any card to save it.
- **Drop Anime**: Click the Ban (slash) icon to hide an anime from your feed.
- **Watch**: Click the "Check Anime" link to instantly search for the anime on Aniwatch.
